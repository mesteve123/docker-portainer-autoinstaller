# Docker & Portainer Auto Installer

Install Docker Engine and Portainer CE with a single script.

## Features
- Works on **Ubuntu 20.04 / 22.04 / 24.04**
- Portainer starts automatically on system boot
- Secure access to Portainer via **HTTPS (port 9443)**

## Installation
```bash
git clone https://github.com/mesteve123/docker-portainer-autoinstaller.git
cd docker-portainer-autoinstaller
chmod +x install_docker_portainer.sh
sudo ./install_docker_portainer.sh
